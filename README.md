# OrangeHRM Recruitment Automation

This project contains a Selenium WebDriver automation script in Java to test the 'Recruitment Module' of [OrangeHRM]. 
The test script verifies that an admin user can add and search "Vacancies" and "Candidates" successfully.

## User Story

> As an admin user, I should be able to add "Candidates" and "Vacancies" and search the added "Candidates" and "Vacancies".

## Test Steps

The Selenium test script performs the following steps:
1. Login to [OrangeHRM] as Admin.
2. Navigate to the 'Recruitment Module'.
3. Add two "Vacancies".
4. Search the added "Candidates".
5. Add two "Candidates".
6. Search the added "Vacancies".
